# PPT Extract

Use the mineru tool to extract content from legacy .ppt PowerPoint files (PowerPoint 97-2003 binary format). Extract text, images, tables, shapes, speaker notes, and metadata from each slide. Output structured content preserving slide order and hierarchy. Handle embedded objects and complex layouts.
